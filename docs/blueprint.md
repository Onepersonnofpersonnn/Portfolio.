# **App Name**: KingDev Portfolio

## Core Features:

- Homepage Section: Display an engaging introduction as 'John King - Professional Web Developer' with a brief summary.
- Skills Showcase: Dedicated area to list technical skills like JavaScript, Python, HTML, and CSS with proficiency level indicators.
- Projects Display: Gallery to present completed web development projects, including project descriptions and technology used.
- Contact Form: Functional form for visitors to send direct messages.

## Style Guidelines:

- Primary color: Vivid blue (#4287f5) for trustworthiness and reliability as well as catching the user's attention.
- Background color: Light desaturated blue (#E8F0FE) to complement the primary blue and provides a softer visual base.
- Accent color: A saturated purple (#A64AC9) which is analogous to blue, creates a vibrant and energetic secondary highlight color.
- Body and headline font: 'Inter' sans-serif font that works well for headlines and body text.
- Clean, single-page layout with clear sections for introduction, skills, projects, and contact.
- Use minimalistic icons to represent different skills and technologies.
- Subtle hover animations on project thumbnails and skill icons to add interactivity.
